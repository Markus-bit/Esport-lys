# E-Sport Lightsystem

Light configuration website for the E-sport room


## Used By

This project is used by the following team(s):

- E-sport Elvebakken


## Features

- CMS updated pages
- Blog / Content posts
- Roster view controlled by CMS
- Mobile friendly


## Tech Stack

**Client:** React, TailwindCSS, styledComponents

**Database:** Sanity

**Server:** Node, Express

**Host:** Firebase
